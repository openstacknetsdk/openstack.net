IIdentityProvider Methods openstack.net API Reference Documentation

[This is preliminary documentation and is subject to change.]

The IIdentityProvider type exposes the following members.
Methods
  NameDescription
AddUser
Adds a user to the account.
Authenticate
Authenticates the user for the specified identity.
Remarks
NOTE: This method always authenticates to the server and never from cache.
DeleteUser
Deletes the specified user from the account.
GetRolesByUser
Gets all roles associated to the specified user
GetToken
Gets the authentication token for the give user.
Remarks
NOTE: Unless overwritten, this will first check the cache for existance and expiration before checking for fresh data from the server
GetTokenInfo
Gets the full IdentityToken details.
Remarks
NOTE: Unless overwritten, this will first check the cache for existance and expiration before checking for fresh data from the server
GetUser
Retrieves the details for the specified user.
GetUserAccess
Gets the user access details.
GetUserByName
Gets the details for the specified username
GetUserCredential
Gets the specified user credential.
ListTenants
Lists the tenants for the user.
ListUserCredentials
Lists the credentials for the specified user.
ListUsers
Lists all the User for the account.
UpdateUser
Updates the details for the specified user.
